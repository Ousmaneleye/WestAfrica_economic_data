# WestAfrica_economic_data

## Clone and run the program
```bash
git clone
cd WestAfrica_economic_data/
source bin/activate
```
### Install requirements
```bash
pip3 install -r requirements
```
### Run mat_plot.py
```bash
cd sources/
python3 mat_plot.py
```
